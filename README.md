# CLOYD-SWARM 🤖🌐

**Multi-Agent AI Swarm Framework - Fully Offline, 100% Autonomous.**

CLOYD-SWARM adalah sistem AI dual-engine yang menggabungkan kecepatan **C++ Native Engine** dengan kecerdasan **Python Autonomous Scavenger**. Sistem ini memungkinkan AI untuk menjelajah internet secara mandiri menggunakan agen swarm dan menyuntikkan pengetahuan tersebut ke dalam model lokal (Ollama).

## ✨ Fitur Utama
- **C++ Native Brain:** Engine chat super ringan tanpa beban library JSON eksternal.
- **Autonomous Scavenger:** Agen Python yang menjelajah internet secara otomatis.
- **Multi-Agent Swarm:** Mendukung hingga 20 agen pencari informasi secara paralel.
- **Knowledge Injection:** Sistem RAG (Retrieval-Augmented Generation) sederhana melalui file teks.
- **Fully Offline:** Berjalan sepenuhnya di mesin lokal menggunakan Ollama.
